# Scottish Power Compliance Playbook

## Known aliases
- Scottish Power, ScottishPower, SP

## Supplier-specific rules

1. **TPI (third-party intermediary) disclosure.** Agent must clearly
   state they are an independent broker and NOT Scottish Power.

2. **Multi-site contracts.** When the deal spans multiple sites, the
   agent must list or reference all sites included in the contract.

3. **Rate structure disclosure.** Agent must mention standing charge,
   unit rate, AND any day/night differentials if applicable.

## Customer confirmation

Standard list applies. Scottish Power customer confirmations are often
brief — "yep" or "aye" alone is valid confirmation.
