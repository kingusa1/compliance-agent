# Pozitive Energy Compliance Playbook

## Known aliases
- Pozitive, Pozitive Energy

## Supplier-specific rules

1. **Spelling confirmation.** The agent MUST ask the customer to spell
   names, addresses, or position titles if there is any ambiguity. If
   agent doesn't ask for spelling, mark `partial`.

2. **Call duration.** Agent must state the call takes "approximately 8
   minutes" (this is a signature Pozitive phrase).

3. **GDPR / T&Cs reference.** Agent must reference "www.pozitive.energy"
   and terms & conditions.

4. **Contract intent phrase.** Agent must say "legally binding verbal
   commercial supply contract" or near-identical phrasing. This is a
   verbatim-leaning phrase for Pozitive even when strictness is mandatory.
